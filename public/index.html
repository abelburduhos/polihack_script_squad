The pipeline

Three stages, one map.

01

Place

Pick a sensor type and radius, click anywhere on the map. The zone is persisted and rendered in real time.

02

Fill

A hex grid of sensors covers the entire area. Each one gets an ID, a position, and a detection radius — no manual placement needed.

03

Stream

Hover any sensor to see live readings. Every metric is persisted to PostgreSQL with full history.
